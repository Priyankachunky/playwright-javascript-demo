# Playwright POM + Data-Driven Framework

This workspace contains a clean Playwright + JavaScript setup using:
- Page Object Model (POM)
- Data-driven test structure
- No sample test files

## Structure
- pages/: reusable page objects
- tests/: your real test specs
- data/: JSON test data
- fixtures/: shared fixtures and test setup
- utils/: helper utilities

## Install
npm install

## Run tests
npx playwright install
npm test
